# Birriería Cócula

Sitio web para Birriería Cócula — birria tradicional en Guadalajara, Jalisco, desde 1978.

Sitio estático (HTML/CSS/JS) hospedado en GitHub Pages.

## Desarrollo local

```bash
python3 -m http.server 4321
# abrir http://localhost:4321
```

## Estructura

- `index.html` · marcado de la página
- `styles.css` · estilos (Fraunces + Instrument Serif + DM Mono)
- `main.js` · tabs del menú, navegación móvil, reveals
- `assets/images/` · fotos optimizadas
